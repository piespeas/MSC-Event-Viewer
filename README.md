# MSC Event Viewer

<hr>

This tool lets you view various event IDs required for screensharing.

It is built with **Electron** and is open-source.

It *may* send a virus warning due to it not having a `WINDOWS APPLICATION LICENSE` (100$ fee).

The code is **Far from organized well**, Improvements can always be suggested
